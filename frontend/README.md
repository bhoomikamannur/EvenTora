# Eventora Frontend

React frontend for Eventora college communities platform.

## Quick Start
```bash
npm install
# Create .env file with REACT_APP_API_URL
npm start
```

## Available Scripts

- `npm start` - Start development server
- `npm run build` - Build for production
- `npm test` - Run tests

## Environment Variables

Create `.env` file:
```
REACT_APP_API_URL=http://localhost:5000/api
```

## Features

- Responsive design
- Real-time updates
- Authentication flow
- Club management
- Event calendar
- Media gallery
